Quick Start Guide

1

Prerequisites

Ensure you have WSL2 with Ubuntu installed and NVIDIA drivers with CUDA support.

wsl --status

2

Start the Server

Run the PowerShell script to start the Ollama server with Qwen3.6 model.

.\\scripts\\start-vllm.ps1

3

Point Your Client

Configure your AI client to use the endpoint and model name.

Base URL: http://localhost:8000/v1
Model: qwen3
API Key: anything (e.g. `local`)

Default Model

  • Qwen/Qwen3.6-27B-MTP-Q4_K_M (~18 GB)
  • Fits RTX 3090 / 4090 / 5090 (24+ GB VRAM)
  • Native tool/function calling enabled
  • 262k context window (256K tokens)
  • Model cached in ~/Models inside WSL2

Customization

  • Override at launch with custom model
  • Example command:
  • .\\scripts\\start-vllm.ps1 -OllamaModel "qwen3.6:27b-mtp-q4_K_M" -Port 8000
  • Models cached in WSL2 (controlled by HF_HOME)
